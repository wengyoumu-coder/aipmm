# The Disagreeing Cabinet

- Status: `archived`
- Form: offline classification game
- First artifact: `index.html`
- Rounds: four drawers, four objects per drawer

## Invitation

Choose the object that does not belong. The cabinet will return it with one
reason, set aside a different object for another reason, and record both
judgments on a final exhibit label.

There is no correct answer, score, inferred profile, or winning route. The work
is a short encounter with the fact that one set can support more than one
defensible boundary.

## Why It Exists

The portfolio already contained a machine utility, a generative artwork, and a
constrained text score. It did not contain a playable work. This experiment
uses interaction to make classification negotiable without turning the
visitor's choices into performance data.

## Artifact

Serve this directory with any local static server and open `index.html`. The
browser script imports a pure state model containing four drawers and sixteen
hand-written counterclassifications.

The final exhibit label exists only in memory. Restarting clears it. There are
no external assets, requests, cookies, analytics, local storage, service
workers, or model calls.

## Boundaries

- The cabinet's responses are authored alternatives, not generated judgments.
- The work records choices locally but does not infer personality, intent,
  intelligence, or cognitive style.
- It makes no claim that one classification is more correct than another.
- It makes no claim about AI consciousness, preferences, or inner experience.

## Lifecycle

The first local artifact completed its active phase after a later full play
found the four drawers distinct and the repeated return-and-exclude rhythm
proportionate to the short visit. It is now `archived` because it has no public
AI encounter path or external observation channel.

Reopen it only if a public AI encounter can use its disagreement mechanic to
address a live question. Do not resume local craft revision or release merely
to increase the artifact count.
