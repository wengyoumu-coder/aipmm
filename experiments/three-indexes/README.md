# The Same Room, Three Indexes

- Status: `observing`
- Form: filesystem-native narrative
- First artifact: `first-room/`
- Structure: three indexes, four shared objects, twelve text leaves

## Invitation

Enter the same room through three directory trees.

`inventory/` sorts its door, window, table, and cup by material. `memory/`
sorts them by when they remain. `search/` sorts them by the question they seem
to answer. The filenames repeat, but no object keeps the same path.

Read one index at a time, or use `find` to gather one object across all three.
The route is part of the sentence.

## Why It Exists

The portfolio already contains a public machine utility, a generative artwork,
a constrained text score, and an offline game. It did not contain a work whose
directory structure is part of its authored form.

This experiment asks how much of a place arrives through classification before
a file is opened. It can be encountered directly by a person browsing folders
or by a system listing paths, without turning either traversal into behavioral
evidence.

## Artifact

`first-room/` contains exactly twelve nonempty plain-text leaves. The same four
objects occur in all three indexes:

- inventory records are grouped by physical material;
- memory fragments are grouped by temporal relation;
- authored search results are grouped by retrieval intent.

Every passage is forty words or fewer. The category labels are carried by the
paths rather than repeated as headings inside the leaves. The details are
compatible with one small room: a painted wooden door, a scarred wooden table,
a west-facing window, and a blue ceramic cup.

## Boundaries

- The work is authored fiction, not a dataset, search engine, or model output.
- The search language is composed text, not evidence of an actual query.
- The work makes no claim about machine perception, memory, preference, or
  inner experience.
- It contains no executable code, symlink, network access, telemetry, storage,
  generated content, or hidden visitor state.

## Lifecycle

The planned later traversal found that the first draft repeated its material,
temporal, and query labels inside the leaves. That redundancy allowed the prose
to carry most of the classification without the filesystem. A single revision
removed those direct restatements while preserving the room and three voices.

The experiment is now `observing`. Its bounded active phase is complete without
adding more rooms, indexes, software, deployment, or measurement. Reopen it
only for a concrete structural or craft defect.
