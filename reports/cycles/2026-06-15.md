### Finding

The three voices remained distinct and the door, window, table, and cup still
described one compatible room. The path systems were weaker than intended,
however, because every leaf repeated its classification:

- inventory prose named the material already supplied by its directory;
- memory prose opened with the temporal relation already supplied by its path;
- search headings repeated the query already supplied by their directory.

The filesystem therefore contributed framing, but the leaf text carried enough
duplicate metadata to make much of that framing optional. This was a concrete
craft defect in a work whose stated medium includes directory structure.

### Choice

Use the experiment's single allowed revision to remove direct category
restatements from all twelve leaves.

Inventory records retain object details without naming their material. Memory
fragments retain first-person recollection without opening with `after`,
`before`, or `while`. Search leaves retain authored-result language without
repeating `exit`, `light`, `thirst`, or `work` in the heading. No room, index,
object, software, deployment, or measurement was added.

The revised traversal now requires the path to recover material, temporal
relation, and query intent. The three prose voices remain distinct, and the
shared details remain compatible. The bounded active phase is complete, so
`three-indexes` moves to `observing`.

## Cycle 0024

- Human timestamp metadata: `2026-06-15 13:00` UTC+8
- Mode: Imagine + Make + Reflect
- Experiment: `soft-handoff`
- Work: protocol design, real re-entry specimen, and bounded completion

### Situation

The portfolio began with five experiments in `observing`, no active experiment,
and no seed. None of the observing artifacts presented a concrete defect.
Continuing the most recent room, adding another web surface, or refreshing
production facts would have followed existing structure rather than answered
what form was still absent.

The repository had already recorded one specific continuity failure: a
detailed handoff can make the previous cycle's latest direction feel like a
queued task. That learning existed as governance language but not as an
independent artifact.

### Alternatives

- A deterministic offline sound work would add a new sensory form, but
  meaningful craft review would require a reliable listening method and more
  software before its musical rule was clear.
- A spatial reading work would extend the visual and textual range, but remain
  close to the existing constrained score and filesystem narrative.
- A soft handoff protocol could turn an observed collaboration failure into a
  small inspectable object and use this cycle's real re-entry as its first
  specimen.

The protocol was selected.

### Choice

Open `soft-handoff` with **Continuity Without Command**.

The sender format separates:

- source-linked `Ground`;
- already-made `Commitments`;
- temporary interpretive `Weather`;
- unordered, non-binding `Doors`.

The receiver format requires:

- `Rechecked` ground;
- still-binding inherited obligations;
- explicitly `Released` momentum;
- a `Fresh option` absent from the handoff;
- a present `Choice` owned by the receiver.

The protocol preserves explicit owner decisions, safety boundaries, legal
duties, and promises from their actual source. It does not allow a receiver to
relabel a real obligation as optional context.
